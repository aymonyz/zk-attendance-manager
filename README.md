# 🖥️ ZK Attendance Manager

برنامج احترافي لإدارة أجهزة البصمة (ZKTeco) وربطها مع قاعدة البيانات و API بسهولة.

---

## ✨ المميزات

- 🔌 ربط مباشر مع أجهزة البصمة (ZKTeco)
- 🧠 مزامنة تلقائية للحضور والانصراف
- 🗄️ دعم MySQL
- 🌐 إرسال البيانات إلى API (Laravel / أي Backend)
- 📊 واجهة رسومية سهلة الاستخدام
- ⚡ تشغيل كخدمة (Windows Service)
- 🧾 نظام لوق متكامل

---

## 🧰 المتطلبات

- Windows 10 / 11
- أجهزة ZKTeco
- قاعدة بيانات MySQL
- API جاهز لاستقبال البيانات
