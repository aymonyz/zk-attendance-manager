# 🖥️ ZK Attendance Manager

برنامج احترافي لإدارة أجهزة البصمة (ZKTeco) وربطها مع قاعدة البيانات و API بسهولة![IMG-20250309-WA0003](https://github.com/user-attachments/assets/69e5e261-5b9a-402f-9185-9241e9738331)

---

## ✨ المميزات

- 🔌 ربط مباشر مع أجهزة البصمة (ZKTeco)
- 🧠 مزامنة تلقائية للحضور والانصراف
- 🗄️ دعم MySQL
- 🌐 إرسال البيانات إلى API (Laravel / أي Backend)
- 📊 واجهة رسومية سهلة الاستخدام
- ⚡ تشغيل كخدمة (Windows Service)
- 🧾 نظام لوق متكامل

---

## 🧰 المتطلبات

- Windows 10 / 11
- أجهزة ZKTeco
- قاعدة بيانات MySQL
- API جاهز لاستقبال البيانات
